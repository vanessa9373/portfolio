## About This Repo

This is the source code for my personal portfolio website, live at **[vanessa9373.github.io/portfolio](https://vanessa9373.github.io/portfolio/)**.

Built with HTML, CSS, and JavaScript — fully responsive, dark-themed, with smooth animations, skill bars, project cards, and a contact form.
